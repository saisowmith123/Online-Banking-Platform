# Online Banking Platform

A full-stack online banking application that lets users manage accounts, 
view transaction history, and transfer funds in real time.

## Features
- User registration and login (JWT-based authentication)
- View account balance and transaction history
- Open new accounts
- Transfer money between accounts
- Deposit and withdraw funds
- Real-time balance updates on the dashboard

## Tech Stack
**Frontend:** React, Redux, Material UI  
**Backend:** Spring Boot, REST API  

## How to Run

### Backend
- mvn spring-boot:run

### Frontend
- npm install
- npm start

## What I Built
- Designed and implemented REST APIs for account management, transfers, 
  deposits, and withdrawals
- Built the React/Redux frontend with real-time state updates
- Connected frontend and backend with secure JWT authentication
